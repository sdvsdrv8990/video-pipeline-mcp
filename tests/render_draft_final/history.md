# История теста: render_draft_final

## Статус: РАБОЧИЙ ТЕСТ

## Назначение
Проверка полного пайплайна рендера: draft → final через реальные инструменты.

## Тип теста
Workflow / Integration (тест инструментов)

## Что тестируем
1. Создание структуры проекта (fs_create_file)
2. Draft рендер (создание снапшота)
3. Final рендер (привязка к draft)
4. Верификация через fs_read_file

## Зачем нужен
1. **Рабочий пайплайн:** Draft → final основной кейс
2. **Инструменты:** Проверка fs_* инструментов
3. **Цепочка:** Проверка связей между рендерами

## Что хотим увидеть
- Draft рендер создаётся успешно
- Final привязан к draft через derived_from_render_id
- Все файлы доступны через fs_read_file
- Facts содержат информацию о рендерах

## Как отражает реальное поведение
Эмулирует реальный пайплайн создания видео: от сценария до финального рендера.
