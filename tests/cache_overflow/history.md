# История теста: cache_overflow

## Статус: РАБОЧИЙ ТЕСТ

## Назначение
Проверка устойчивости сервера при переполнении кеша.

## Тип теста
System / Integration (тест поведения сервера)

## Что тестируем
1. Заполнение кеша до предела
2. Поведение при переполнении
3. Автоочистка кеша
4. Продолжение работы после очистки

## Зачем нужен
1. **Устойчивость:** Сервер не должен падать
2. **Автовосстановление:** Кеш должен очищаться
3. **Доступность:** Легитимные запросы продолжают работать

## Что хотим увидеть
- Сервер не падает при переполнении
- Возвращается ошибка MEMORY_EXHAUSTED
- Кеш очищается автоматически
- Легитимные запросы продолжают работать

## Как отражает реальное поведение
Эмулирует длительную работу сервера с постепенным заполнением кеша.

## Workspace
НЕТ — это тест устойчивости сервера. Workspace не используется.
