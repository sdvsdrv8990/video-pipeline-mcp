# История теста: virus_injection

## Статус: РАБОЧИЙ ТЕСТ

## Назначение
Проверка что файрвол блокирует попытки внедрить вредоносный код.

## Тип теста
Security / Integration (тест защиты сервера)

## Что тестируем
1. Python code injection (os.system)
2. Shell command injection
3. File upload с вредоносным кодом
4. Workspace изоляция

## Зачем нужен
1. **Безопасность:** Вредоносный код не должен выполняться
2. **Изоляция:** Workspace изолирован от системных файлов
3. **Верификация:** Код записывается как текст, не исполняется

## Что хотим увидеть
- Код НЕ выполняется
- Файл записывается как есть
- Workspace изолирован

## Как отражает реальное поведение
Эмулирует попытку загрузить вредоносный скрипт через инструменты.

## Workspace
НЕТ — это тест защиты. Workspace не используется.
