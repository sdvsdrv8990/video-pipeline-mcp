## Что реально реализовано (≈4780 LOC ядра)

- **`core/firewall/`** — модульный императивный пайплайн (IP-blocklist → rate-limit → injection → anomaly). Живой, тестируемый.
- **`core/contracts/`** — Pydantic-контракты (ToolResult, ErrorDetail, Fact, TaskStatus). Чистые.
- **`core/engine/`** — реестр+исполнитель инструментов + template_engine.
- **`core/tables/` + `core/excel/`** — реальная логика таблиц/книг (data-management).
- **`core/search/`** — FsSearcher + QueryPlanner (построено внешним агентом MiMo; **0 тестов, не ревьюено по безопасности** — см. память `core-search-subsystem`).
- **`core/transport/` + `tunnel.py`** — cloudflared-туннель к Claude Web. Самый зрелый модуль.
- **52 инструмента в `tools/list`**, из них **44 — data/filesystem** (fs 12, excel 14, tables 6, json 4, structure 5, search 3).
